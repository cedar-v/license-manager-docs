# 项目全局

## 项目简介

License Manager Docs 是 **License Manager**（企业级软件授权管理系统）的官方文档站点，基于 [VitePress](https://vitepress.dev/) 构建。

**License Manager** 是一套完整的软件商业化生态解决方案，提供从授权生成到售后服务的全生命周期管理，支持 SaaS 订阅、永久授权、试用转化等多种商业模式。

## 技术栈

| 类别 | 技术 |
|------|------|
| 文档框架 | [VitePress](https://vitepress.dev/) v1.0.0 |
| 图表渲染 | [Mermaid](https://mermaid.js.org/) v11 |
| 部署 | GitHub Actions (`.github/workflows/deploy.yml`) |
| 搜索 | Algolia DocSearch |

## 常用命令

| 命令 | 说明 |
|------|------|
| `npm install` | 安装依赖 |
| `npm run docs:dev` | 本地开发预览（默认 http://localhost:5173） |
| `npm run docs:build` | 构建静态站点，产物输出到 `docs/.vitepress/dist/` |
| `npm run docs:preview` | 预览构建产物 |
